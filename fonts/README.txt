Place Montserrat-ExtraBold.ttf in this folder.

Download from Google Fonts:
  https://fonts.google.com/specimen/Montserrat

1. Click "Download family"
2. Unzip the archive
3. Find: static/Montserrat-ExtraBold.ttf
4. Copy it here: influencer-automation/fonts/Montserrat-ExtraBold.ttf

The thumbnail processor will fall back to a basic system font if this file
is missing, but the result will look much worse.
